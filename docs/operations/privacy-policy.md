# ZironSec Privacy Policy

## Introduction
ZironSec is committed to protecting visitor and customer data in accordance with privacy-by-design principles, GDPR, and CCPA.

## Data We Collect
- Personal contact information voluntarily submitted via secure form.
- Non-identifiable analytics data (page views, session time) through permitted providers.

## Data Use
- Respond to requests and provide security services.
- Improve security posture and user experience.

## Sharing and 3rd Parties
- We only share data under contract (DPAs) with trusted processors for calendaring and support workflows (e.g., Calendly, secure email).

## Retention and Deletion
- Personal data retained only for as long as required. Subjects may request deletion via support@zironsec.com.

## Controls
- Access control: AC-3 (role-based access enforced on internal systems)
- Security: SI-7 (integrity validation), SC-8 (encrypted transport)
